In this experiment, I want to use this like button to collect user's feedback. Like buttons are widely used on lots of websites, like facebook and youtube. This is the simplest way to get feedback from users. Especially for lazy guys~ :)

I have to say, I drew this thumb up pic and changed color for it in order to match my new homepage. Making my own icons was fun, but took me a lot of time...


First, I made hover for this img.

<img id="likeButton" class="likeimg" src="../../images/image/like.png" onmouseover="this.src='../../images/image/likehover2.png'" onmouseout="this.src='../../images/image/like.png'" />

Use mouseover and mouseout to change different images for my button.

When you clicking, my function likecheckHandler was actived.

$("#likeButton").click(likecheckHandler);

Instead of change image in the function, I added a marker to mark the status of clicking. If this is your first clicking, means you want to submit your like, I would add a class named liked to the image. If you've clicked once, you want to cancel your like, I would check the class then remove it.

function likecheckHandler() { if ($("#likeButton").hasClass("liked") == false) { var mark = "Liked<br />Thank you<br />You could cancel your like by clicking again"; $("#likeButton").addClass("liked"); checkstatus(mark); } else { var mark = "removed like<br />You could relike by clicking the button"; $("#likeButton").removeClass("liked"); checkstatus(mark); } checkimg(); }

Notice, I called function checkstatus() after changing the class.

function checkstatus(mark) { $("#likestatus").empty(); $("#likestatus").append(mark); }

This function is used to change the introduction below the thumb-up image, which gives the status as well. I pass the variable mark into it.


At last, I need to change the image when you added the like. I have the class liked which marks the right time to show like and change another image to give reponse to users.

function checkimg() { if ($("#likeButton").hasClass("liked") == true) { document.getElementById("likeButton").src = "../../images/image/liked2.png"; } else document.getElementById("likeButton").src = "../../images/image/like.png"; }
