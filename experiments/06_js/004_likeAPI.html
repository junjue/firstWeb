Here I added some famous like or share APIs into my experiment. Facebook and Twitter are more famous and popular social methods obviously, using their's API is a convenient way to share with friends.

First I tried Facebook's API. I use a div to contain the frame.

<div class="fb-like" data-href="https://developers.facebook.com/docs/plugins/" data-layout="standard" data-action="like" data-show-faces="true" data-share="true"> </div>

Then use javascript to make it alive. This might cost several seconds to lode.

!function (d, s, id) { var js, fjs = d.getElementsByTagName(s)[0], p = /^http:/.test(d.location) ? 'http' : 'https'; if (!d.getElementById(id)) { js = d.createElement(s); js.id = id; js.src = p + '://platform.twitter.com/widgets.js'; fjs.parentNode.insertBefore(js, fjs); } }(document, 'script', 'twitter-wjs');

Then I tried Twitter's API. Added an anchor, you could click it to share on twitter. Put in a div as well.

<a href="https://twitter.com/share" class="twitter-share-button">Tweet</a> <div id="fb-root"></div>

Here below is the script of Twitter's API.

(function (d, s, id) { var js, fjs = d.getElementsByTagName(s)[0]; if (d.getElementById(id)) return; js = d.createElement(s); js.id = id; js.src = "//connect.facebook.net/en_US/sdk.js#xfbml=1&appId=313485655511294&version=v2.0"; fjs.parentNode.insertBefore(js, fjs); }(document, 'script', 'facebook-jssdk'));

From this experiment, I learnt how to include Facebook and Twitter's API to let users to share my website.

This is different with the other experiment about like button, which is simply by clicking the thumb-up button.

They would be used in different situations.
