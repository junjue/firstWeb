I want to show images changing automatically on my website, so I use javascript to achive it.

There are three functions in my script. One is the timer of changing, the other two are controller of buttons.

<script type="text/javascript"> function dispNext() {...} function dispPrev() {...} function startTimer() {...} </script>

Set interval timer to change images as 1500ms by using dispNext().

<script type="text/javascript"> function startTimer() { setInterval(dispNext, 1500); } </script>

Change the value of document.getElementById in order to change pictures. Use x + 1 or x - 1 to decide next/prev image to show by clicking buttons.

<script type="text/javascript"> function dispNext() { x = (x === images.length - 1) ? 0 : x + 1; document.getElementById("img").src = images[x]; } function dispPrev() { x = (x <= 0) ? images.length - 1 : x - 1; document.getElementById("img").src = images[x]; } </script>
