Use jQuery-ui and learn to use draggable and sortable attributes.

I tried sortable lists by using .sortable(). You could reorder lists by dragging up and down.

<div class="myfont"> <ul> <li id="draggable">Add new</li> </ul> <ul id="sortable"> <li>Diana</li> <li>Peter</li> <li>Nilson</li> <li>Derek</li> </ul> <hr /> </div>

Added a new item above, you could drag it down to the list which clones into the list by the order you put.

$(function () { $("#sortable").sortable({ revert: true }); $("#draggable").draggable({ connectToSortable: "#sortable", helper: "clone", revert: "invalid" }); $("ul, li").disableSelection(); });
