This experiment is to learn how to use button to scroll up to the page's top. The scroll up button would show when scroll down the page.

anchor of scroll up button
<a href="#" class="scrollToTop"></a>

I made this image and used it as the scroll up button.

Here I set the position and url of the image in ../css/myStyle.css file. When mouseover, the picture would zoom out.

../css/myStyle.css
<style> .scrollToTop { width: 10%; height: 20%; position: fixed; bottom: 35px; right: 35px; display: none; background: url('../../images/image/upsign.png') no-repeat 0px 20px; } .scrollToTop:hover { zoom: 110%; } </style>

I use jquery to controll this scroll to top button.

First to check if the window is top. If not, trigger the button to show.

../js/scrollTop.js
$(window).scroll(function () { if ($(this).scrollTop() > 100) { $('.scrollToTop').fadeIn(); } else { $('.scrollToTop').fadeOut(); } });

Then, when the button shows, you could go back to the top of the page by click the button.

Return to the page's top by using {scrollTop: 0}.

../js/scrollTop.js
$('.scrollToTop').click(function () { $('html, body').animate({ scrollTop: 0 }, 600); return false; });
