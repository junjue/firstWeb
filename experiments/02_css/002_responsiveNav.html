Here is a responsive navigation menu. You could change the size of window, when the width is smaller than 480px, the menu would become a button. If you click the button, the menu would show at your left hand side.


This nav bar is designed for my homepage. If users use cellphone or tablet, the page could shrink in a convenient way. If you need to navigate to other routh, you could find the button easily.

Code:

Use max-width to control the maximum window size I want to shrink the menu. Control the elements and the positions of them.

@media handheld, screen and (max-width: 480px) { .smenu-toggle, .s-menu-toggle-button, .s-toggleable-menu { top: 0; position: fixed; } .smenu-toggle { left: 0; margin: 0; display: inline-block; width: 15%; z-index: 999; }

Check the position of toggled menu, make sure it attached appropriately to the left.

.s-toggleable-menu.s-left { left: -105%; } .s-menu-toggle-button:checked ~ .smenu-toggle.s-left { left: auto; right: 0; } .s-menu-toggle-button:checked + .s-toggleable-menu.s-left { width: 85% !important; left: 0; } }
