Orginize Code

Since I was from Level Zero for web development, I did not have an overall view when I was coding. Luckily, from now on, I've learned how to version control my code via Git and Github. So I began to re-organize my code, pushed them to Github. I've also learned how to use branch to safely update code then finally merge to my master branch. This is really awesome since previously, I need to copy my files to other folders as backup. Now all I need is to create a new branch, commit or reset my HEAD to any version I want.
Git commands learnt:
git checkout -B new-branch-name: create new branch
git merge branch-name: merge branch to current branch
git reset HEAD: reset current changes to previous HEAD
git checkout -- file-name: reset changed file content
