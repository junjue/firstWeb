Project Design

While designing my project, I feel myself still lack of a general idea on how the framework works for a website. Moreover, I still need more knowledge on this. Based on this, I've read some materials on the structures and frameworks. To my understanding, since we should use MVC on both server side and client side, I've decided to build my project with the following framework:
    On the server side:
  • Model: all the models and MongoDB
  • View: Express, NodeJS
  • Controller: NodeJS, Express
    On the client side:
  • Model: Javascript
  • View: CSS, JADE
  • Controller: Javascript
